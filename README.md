Live Code Editor

مشروع محرر أكواد بسيط يعمل داخل المتصفح، يتيح كتابة أكواد HTML وCSS وJavaScript في نصف الشاشة، مع عرض النتيجة بشكل فوري في النصف الآخر.


فكرة المشروع قائمة على التعلم العملي وتجربة الكود مباشرة دون الحاجة لأي إعدادات أو أدوات خارجية، بهدف تبسيط فهم كيفية تفاعل HTML وCSS وJavaScript معًا داخل المتصفح.


المحرر موجه للمبتدئين والمهتمين بتجربة الأفكار بسرعة، ويشكّل أساسًا قابلًا للتطوير لاحقًا ليصبح أداة تعليمية أو محررًا أكثر تقدمًا.


## MVP Status

This project is currently in its MVP (Minimum Viable Product) stage.

It provides the core functionality:
- Write HTML, CSS, and JavaScript
- See the result instantly in the browser

All additional features will be considered after validating and stabilizing this core experience.
